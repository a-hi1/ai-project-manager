# 🚀 项目管理系统

基于 **Spring Boot + Vue 3 + Element Plus + PostgreSQL + AI** 的全栈项目管理系统

---

## ✨ 功能特性

### 核心功能
- 📋 **项目管理** - 项目创建、编辑、成员管理
- 🎯 **任务管理** - WBS任务分解、甘特图、看板视图
- 📊 **需求调研** - 需求收集与分析
- 🔬 **可行性分析** - 技术、经济、市场分析
- ⚠️ **风险管理** - 风险识别、评估、应对
- 🐛 **缺陷管理** - Bug跟踪与修复
- 🔄 **变更管理** - 变更请求与审核
- 📦 **交付验收** - 项目交付与验收
- 📁 **文档归档** - 项目文档管理
- 🔍 **项目复盘** - 项目总结与经验积累
- 📈 **结束报告** - 项目结束报告生成

### AI智能辅助
- 💬 **AI对话助手** - 智能问答 + 文档上传分析
- 🧠 **需求解析** - 智能需求分析 + 文档上传解析
- 📝 **任务拆分** - AI自动拆分任务
- 📚 **知识库** - 项目知识管理 + RAG检索
- 🏥 **健康度分析** - AI分析项目健康度
- 📋 **生成日报** - 自动生成项目日报
- 📌 **任务建议** - AI提供任务建议
- 📝 **项目总结** - 自动生成项目总结

### 文件上传支持
系统支持多种文件格式的上传和自动解析：
- 📄 **文档类型**: .txt, .md, .rst, .rtf
- 📊 **Office文档**: .docx, .doc, .xlsx, .xls, .pptx, .ppt
- 💻 **代码文件**: .js, .ts, .py, .java, .go, .cpp, .c, .h
- 🌐 **Web文件**: .html, .css, .json, .xml, .yaml, .yml
- 📈 **其他**: .csv, .log, .sql, .ini, .cfg, .conf, .sh, .bat, .ps1, .pdf

### 系统功能
- 👥 **多角色权限** - 7种角色权限体系
- 📋 **操作日志** - 完整的操作记录
- 🔔 **实时通知** - WebSocket实时推送

## 📥 快速开始

### 前置要求
- Node.js 16+
- Java 17+
- PostgreSQL 14+
- Maven 3.8+

### 1️⃣ 数据库初始化

```bash
# 连接到 PostgreSQL
psql -U postgres

# 创建数据库
CREATE DATABASE project_management;
\c project_management;

# 设置编码（重要！）
SET CLIENT_ENCODING TO 'UTF8';

# 执行初始化脚本
\i d:/zhuomian/软件项目管理/src/main/resources/init.sql;

# 导入演示数据
\i d:/zhuomian/软件项目管理/src/main/resources/demo-data.sql;
```

### 2️⃣ 启动后端

```bash
# 使用 Maven 启动
cd d:/zhuomian/软件项目管理
mvn clean spring-boot:run
```

或者直接双击：
- Windows用户：`start.ps1` 或 `start.bat`
- Linux/Mac用户：`start.sh`

### 3️⃣ 启动前端

```bash
cd d:/zhuomian/软件项目管理/frontend

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### 4️⃣ 访问系统
- 前端地址：http://localhost:5173
- 后端地址：http://localhost:8080

## 🐛 常见问题

### 1. 数据库中文乱码
在执行SQL脚本前先执行：
```sql
SET CLIENT_ENCODING TO 'UTF8';
```

### 2. 后端启动失败
检查 `application.properties` 中的数据库连接信息是否正确

### 3. 前端API请求401
确保已登录，或检查 `localStorage` 中的 `token` 是否有效

### 4. 甘特图不显示
确认项目下有任务数据，或使用演示数据账号（pm）查看
