# Frontend Design Skill

This skill helps improve frontend UI design with professional design principles.

## Core Design Principles

### 1. Typography
- Use clear hierarchy with H1, H2, H3 sizes
- Maintain proper line heights (1.5-1.6)
- Ensure readable font sizes (16px base)
- Avoid excessive font variations

### 2. Color Schemes
- Use max 3 main colors + neutrals
- Maintain proper contrast ratios (WCAG AA)
- Avoid excessive purple gradients
- Use consistent color themes

### 3. Spacing & Composition
- Use 8px grid system
- Maintain proper whitespace
- Avoid deep nesting of cards
- Balance content distribution

### 4. Animations
- Use subtle transitions (200-300ms)
- Avoid excessive motion
- Maintain 60fps performance
- Use easing functions appropriately

### 5. Background Details
- Use subtle patterns or gradients
- Avoid distracting elements
- Maintain readability
- Consider dark/light mode support

## Usage

Invoke this skill when:
- User asks to improve UI design
- User wants to polish existing interfaces
- User mentions "template look" or generic design
- User asks for design suggestions for landing pages, marketing sites, or portfolios
